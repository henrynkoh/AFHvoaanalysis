# How to View AFH Omni-Pulse Landing Page

## Quick Start - View Locally

The development server is now running! 

**Open your browser and go to:**
```
http://localhost:3000
```

You should see the modern, interactive landing page with:
- Sidebar navigation
- Hero section with animated backgrounds
- Features section
- How It Works section
- Insights section
- Case Studies section
- Testimonials section

## If Server Isn't Running

Run these commands in the terminal:

```bash
cd /Users/henryoh/Documents/AFHhub/afh-voa-monitor
npm run dev
```

Then open http://localhost:3000 in your browser.

## Troubleshooting

**Port already in use?**
```bash
npm run dev -- -p 3001
```
Then use http://localhost:3001

**Dependencies not installed?**
```bash
npm install
```

**Build errors?**
```bash
npm run build
```
Check for any error messages.
